# 🚀 QUICK START GUIDE

Get your password manager running in 3 minutes!

## Step 1: Install Dependencies (30 seconds)

Open your terminal/command prompt and run:

```bash
pip install -r requirements.txt
```

Or if using Python 3:
```bash
pip3 install -r requirements.txt
```

## Step 2: Run the Program (5 seconds)

```bash
python password_manager.py
```

Or:
```bash
python3 password_manager.py
```

## Step 3: Create Master Password (1 minute)

1. Program will ask for a master password
2. Choose something strong but memorable
3. **Write it down on paper** and keep it safe!
4. Confirm the password
5. Done! 🎉

## Step 4: Add Your First Password (1 minute)

1. Choose option `1` from the menu
2. Enter site name (e.g., "Gmail")
3. Enter username/email
4. Choose option `2` to generate a strong password (recommended!)
5. Select a category
6. Press Enter to skip notes
7. Your password is now encrypted and saved!
